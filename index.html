Boat Rental Platform · 2024–2025 · whiteislandcharter.com ↗
The Challenge

The client needed a professional boat rental website that could handle complex seasonal pricing — where boat prices change automatically depending on the time of year. The site also had to serve customers across multiple countries, requiring multilingual support in Dutch, French, and English. On top of that, the existing site had slow load times and unresolved security vulnerabilities.

TutorBaba
Ed-Tech Tutoring Marketplace · 2024–2025 · tutorbaba.com ↗
The Challenge

The client needed a full ed-tech marketplace built from the ground up — a platform where students could find and book expert tutors, communicate in real time, and pay securely. The platform had to support multiple user roles (students and experts), handle country-based pricing with automatic currency switching, and send booking confirmations via both email and SMS. After development, the entire site had to be migrated and deployed on AWS.

What I Built

I built TutorBaba from scratch on WordPress + WooCommerce, extending the platform with custom student and expert role logic — each role gets a dedicated dashboard, profile, and booking flow. I integrated real-time chat between students and their assigned tutors using a live chat plugin, and wired up Twilio SMS for instant booking notifications alongside custom-designed email templates.



Country-based pricing was implemented so the platform automatically detects the visitor's country, switches the currency symbol, and applies the correct regional price. Payments are handled via Stripe. For performance, I used Cloudflare + Servebolt / Surfside accelerated hosting for edge caching and speed. The final step was a full AWS migration — configuring EC2, setting up the domain, SSL, and environment variables for production.

NYC Great Movers
Moving Services Platform · 2024–2025 · nycgreatmovers.com ↗
The Challenge

The client needed a complete moving services website built from scratch — with no existing codebase to work from. The site had to allow customers to browse moving packages, book services online, and pay securely. The checkout experience needed to be smooth and fully customized to match the brand, with Stripe as the payment gateway.

What I Built

I built the entire site from the ground up using WordPress + Divi page builder, turning WooCommerce into a service booking platform. Each moving package was set up as a WooCommerce product with custom fields for move size, date, and location. I implemented Stripe payment gateway for secure card processing and customized the checkout flow — removing unnecessary fields, adding service-specific options, and creating a clean confirmation experience.



Performance optimization was applied throughout — compressed images, caching, and clean Divi output — resulting in noticeably faster load times compared to the client's old site.
